# KLR's Tactical TTRPG System v1.0

Created by Kristina Leticia Rodriguez

This system emerged from years of tabletop experience, combining mechanical concepts observed in various homebrew campaigns with original design work. The specific implementation, organization, and many subsystems represent original contributions developed between 2022-2026.

## What is this?

A modular tactical TTRPG system designed for gritty, resource-management-focused gameplay. Features include:

- **Reverse d20 resolution** (lower rolls = success)
- **Speed-based action economy** with big/small action slots
- **Resource degradation** (ammo, armor, magic use dice systems)
- **Escalating injury mechanics** (progressive permanent damage on downs)
- **Skill-based progression** (no traditional levels, exponential XP costs)
- **Survival systems** (food, starvation, risky rest mechanics)

## Content Notes

This system includes mechanics for permanent injury, body horror (loss of limbs/organs), and survival scarcity. Session zero discussions about comfort levels are recommended.
## System Philosophy

This system prioritizes **tactical choices, meaningful consequences, and emergent storytelling** through mechanical scarcity. Combat is fast but dangerous. Resources matter. Bodies accumulate scars that tell the story of survival.
## Design Influences

This system draws on tactical combat traditions, resource management mechanics from survival games, and injury/consequence systems that prioritize narrative impact over instant character death.
## Modularity

Designed to work across multiple settings - cyberpunk, fantasy, sci-fi, or anything in between. Magic, tech, and augment systems adapt to world-building needs.
